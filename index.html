MathChaosEngine

分形与混沌系统可视化引擎

基于MFC构建的数学混沌与生成系统轻量级可视化引擎
本项目旨在通过分形、混沌系统及基于粒子的模拟实时渲染技术，探索数学之美。

详情介绍 GitHub

MFC-Mini-Billiards

基于MFC框架开发的小型交互式台球模拟游戏。

该项目完整实现了消息处理机制、图形设备接口（GDI）渲染技术、基础物理引擎及实时交互功能。
玩家可通过鼠标操控的球杆瞄准并击打主球，系统可精准模拟碰撞效果、自动识别落袋情况，并呈现粒子特效的动态烟花效果。
