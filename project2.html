项目介绍

MathChaosEngine 是一个用于展示分形与混沌系统的可视化项目， 当前实现了分形树生成，并支持参数控制与动态效果展示。

技术实现

  • C++ 图形绘制
  • 递归生成分形结构
  • 随机参数控制自然效果

核心代码


// 示例：分形树递归
void drawTree(float x, float y, float angle, int depth) {
    if (depth == 0) return;

    float x2 = x + cos(angle) * depth * 10;
    float y2 = y + sin(angle) * depth * 10;

    drawLine(x, y, x2, y2);

    drawTree(x2, y2, angle - 0.3, depth - 1);
    drawTree(x2, y2, angle + 0.3, depth - 1);
}

项目收获

通过该项目，我深入理解了递归结构与图形生成的关系， 并提升了对复杂系统可视化的设计能力。
